Natural Language Processing Tools

Writing Recognition

Translates printed documents in image format to electronic documents.

The handwriting recognition program can convert thousands of library books into electronic documents in a short time.

Using python library: pytesseract.

Translation

Auto translation performs translation of one language (source) into another language (destination).

The difficulty of designing an automatic translation program is the ambiguity of the input (multiple meanings).

Using google translate service.

Text Summary

From a long text summarized to a short but still full of content.

Have two main methods: Extractive Summarization and Abstractive Summarization.

Using python library: nltk (Method: Extractive Summarization).

POS-Tagging

Performs the function of separating words and giving words types of words.

Using python library: pyvi (modules: ViPosTagger, ViTokenizer)

Sentitive Analysis

Sentiment analysis is a common NLP task, which involves classifying texts or parts of texts into a pre-defined sentiment

Using python library: nltk (nltk.sentiment.vader: SentimentIntensityAnalyzer).
